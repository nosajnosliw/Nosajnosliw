An App for Out Magazine
  • JCLW

I created the following design as part of a class assignment while interning for Here Media, Inc. The design is an attempt to provide a mobile solution for accessing the Out.com content without the mobile browser. Content would live in an app and would update automatically in accordance with the magazines rss feed. The prototype was initially designed to be programmed in Adobe Flex, but could probably be implemented using HTML5 and Javascript as well.

The home screen would operate in a similar fashion to other news readers like Pulse, allowing the user to swipe through the topics menu horizontally with the three most recent stories from each topic populating the column imediately beneath each section label. Users would then be able to vertically swipe through the stories in each section column.

The entire app would change orientation in response to augmentation of the devices accelerometer.

Upon tapping a story, its contents would be fully accessible by way of a pop out window which could be closed by either tapping the 'outx' button or tapping outside of the pop up box.

If a user wanted to see all of the stories for a particular section, s/he would simply tap the section label to populate the story section with the newest articles from that section only. Tapping the section label again would bring the story section back to its original state with each content column corresponding to the secion title above it.
